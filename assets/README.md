# Brand Assets

## Dot — The DotNetCloud Mascot

<p align="center">
  <img src="logo.png" alt="Dot" width="256" />
</p>

**Dot** is the DotNetCloud mascot — a smiling cloud with a net wrapped around her. Her name completes the project name: **Dot**NetCloud.

### Files

| File | Size | Usage |
|------|------|-------|
| `logo.png` | 1024x1024 | Full-resolution project logo / mascot |

The same asset is used as:
- **README header** — 128px centered
- **Web UI favicon** — multi-size `.ico` (16/32/48px), 32px PNG, 180px Apple touch icon
- **Android app icon** — original source at `src/Clients/DotNetCloud.Client.Android/Resources/AppIcon/appicon.png`
- **Documentation headers** — 64px in CONTRIBUTING.md, Architecture, and API docs

### Usage Guidelines

- Refer to the mascot as **Dot** (she/her)
- Do not distort, recolor, or crop the logo
- When used inline with text, maintain reasonable padding around the image
